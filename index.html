Character creation tutorial
After you download the zip file and extract it, it will contain a template.ase file and 9 textures for each of the looking directions.
If you have Aseprite, you could open template.ase and continue from there. Otherwise, if you don't own it, you can use the nine other textures for reference in some other free art software.

Character requirements
-Must have a white outline.
-Resolution of the textures must be 18x18. (Can be scaled)

When you're done
Once you have all nine looking directions exported into a folder, export it into a zip file and send it to me through my Discord server in #ðŸ¦Š-goober-shooter-feedback.
